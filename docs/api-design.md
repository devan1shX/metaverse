# Api Design 

## AUTH & USER Api

```
GET /api/{role}/users
POST /api/{role}/register
POST /api/{role}/login
GET /api/user/me
DELETE /api/{role}/userid:user_id
```

## Space Api

```
GET /api/{role}/spaces
POST /api/{role}/spaces
GET /api/{role}/spaces/:id
PUT /api/{role}/spaces/:id
DELETE /api/{role}/spaces/:id
```

## Avatar Api

GET /api/{role}/avatars
